### 如何在webpack中配置Typescript

* Typescript是JS的超集   
* 安装两种方法
    ```
    //官方
    npm i typescirpt ts-loader -D
    //第三方开发的 比官方更快
    npm i typescirpt awesome-typescript-loader -D
    ```
* 配置tsconfig.json    
* 配置webpack.config.js   
* webpack4 对应的ts-loader为4以上版本
